## 🧠 Grok 4 系列模型

| 模型 | 定位 | 适合场景 | 特点 |
|---|---|---|---|
| **Grok 4** | 通用模型 | 对话、问答、翻译 | 基础功能完善 |
| **Grok 4 Heavy** | 高性能模型 | 编程、分析、复杂推理 | 强大的推理能力 |
| **Grok 4.1** | 优化迭代版 | 写作、创作、内容生成 | 质量与流畅度优化 |

| 对比项 | Grok 官网 | Grok API 中转站 |
